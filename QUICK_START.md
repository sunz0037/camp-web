# 快速开始指南

## 前置条件

1. Node.js (推荐 v16+)
2. 后端服务已启动（camp-app，默认端口8080）

## 启动步骤

### 1. 安装依赖

```bash
cd camp-web
npm install
```

### 2. 配置API地址（可选）

如果需要修改API地址，创建 `.env` 文件：

```bash
VITE_API_BASE_URL=http://localhost:8080
```

### 3. 启动开发服务器

```bash
npm run dev
```

### 4. 访问登录页面

打开浏览器访问：http://localhost:3000

## 测试登录

### 学生首次登录测试

1. 选择"学生"身份
2. 输入学号（例如：`20240001`）
3. 输入姓名（例如：`张三`）
4. 点击"登录"
5. 如果学号与姓名匹配，会弹出密码设置对话框
6. 输入密码（至少6位，例如：`123456`）
7. 确认密码
8. 点击"确定"完成激活并自动登录

### 学生正常登录测试

1. 选择"学生"身份
2. 输入学号
3. 输入密码
4. 点击"登录"

### 教工登录测试

流程与学生登录相同，选择"教工"身份，使用工号登录。

## 常见问题

### 1. 无法连接到后端

- 检查后端服务是否启动
- 检查API地址配置是否正确
- 检查浏览器控制台是否有CORS错误

### 2. 登录失败

- 检查学号/工号是否正确
- 检查姓名是否匹配（首次登录）
- 检查密码是否正确（正常登录）
- 查看错误提示信息

### 3. 密码设置失败

- 确保密码长度至少6位
- 确保两次输入的密码一致

## 开发说明

- 所有API调用使用原生 `fetch` API，无需额外依赖
- 用户信息和Token保存在 `localStorage` 中
- 登录成功后可以扩展路由跳转到主页
