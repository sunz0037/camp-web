# 前端访问信息

## 开发服务器

### 启动命令

```bash
cd camp-web
npm run dev
```

### 访问地址

**登录页面URL**: http://localhost:3000

### 服务器信息

- **端口**: 3000
- **协议**: HTTP
- **自动打开浏览器**: 是（配置在 vite.config.js 中）

## 页面说明

### 登录页面

访问 http://localhost:3000 即可看到登录页面，包含：

1. **身份选择**
   - 学生
   - 教工
   - 家长/访客（暂未开放）

2. **登录表单**
   - 学号/工号输入
   - 姓名输入（首次登录）
   - 密码输入（正常登录）

3. **功能**
   - 首次登录激活
   - 密码设置
   - 错误提示

## API配置

### 默认API地址

- **开发环境**: http://localhost:8080
- **配置位置**: `src/config/api.js`

### 修改API地址

创建 `.env` 文件：

```bash
VITE_API_BASE_URL=http://your-api-url:8080
```

## 使用流程

### 1. 启动后端服务

```bash
cd camp-app
./gradlew bootRun
```

后端服务地址：http://localhost:8080

### 2. 启动前端服务

```bash
cd camp-web
npm run dev
```

前端服务地址：http://localhost:3000

### 3. 访问登录页面

打开浏览器访问：**http://localhost:3000**

## 测试登录

### 学生首次登录

1. 访问 http://localhost:3000
2. 选择"学生"身份
3. 输入学号（例如：`20240001`）
4. 点击"首次登录？"链接或直接输入姓名
5. 输入姓名（例如：`张三`）
6. 点击"登录"
7. 如果验证通过，弹出密码设置对话框
8. 设置密码（至少6位）
9. 确认密码
10. 点击"确定"完成激活并登录

### 学生正常登录

1. 访问 http://localhost:3000
2. 选择"学生"身份
3. 输入学号和密码
4. 点击"登录"

## 注意事项

1. **后端服务必须运行**：前端需要连接后端API（默认 http://localhost:8080）
2. **CORS配置**：确保后端允许前端域名的跨域请求
3. **浏览器控制台**：如有问题，查看浏览器控制台的错误信息

## 停止服务

按 `Ctrl + C` 停止开发服务器

## 生产构建

构建后的文件在 `dist/` 目录，可以部署到任何Web服务器。
